Файл main_all.py запускает основной проход по сайту XuetangX.
Программа считывает информацию по онлайн-курсам, содержащимся на сайта, а именно:
- ссылка на курс;
- название курса;
- продолжительность курса;
- университет, записавший курс;
- цена на курс;
- наличие бесплатной версии курса;
- количество студентов, обучающихся на курсе.

Данная информация записывается в xlsx файл.

В силу специфики сайта написан дополнительный файл seek_2.py,
который организует более точечный поиск потерянных курсов, которые не были найдены основным проходом.
Данный файл имеет меньшее количество повторных проходов по странице,
а также дает возможность сузить круг поиска, указав первую и последнюю страницы поиска.
Данный файл можно клонировать и запустить параллельно по разным отрезкам.
Перечисленные особенности файла seek_2.py позволяют ускорить процесс поиска потерянных курсов.

При повторном запуске поиска после прерывания обе программы попросят ввести номер страницы, с которой необходимо продолжить поиск.
Также необходимо поменять ссылку на нужную страницу в самом коде.

В процессе работы программы выводят отладочную информацию о текущем цикле, курсе, а также найден ли новый курс.
